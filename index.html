Face Recognition Overview

Our system uses MediaPipe and Python to detect and recognise faces in real time.

Project Summary

MediaPipe detects faces instantly while OpenCV processes the webcam feed. Recognition compares face encodings with stored data.

How It Works

  • MediaPipe detects the face & landmarks
  • OpenCV captures webcam frames
  • Face encodings are extracted
  • Recognition compares new encodings with known ones

BASIC OVERVIEW OF CODE (Real code is in our GitHub)

import cv2 import mediapipe as mp mp_face = mp.solutions.face_detection mp_draw = mp.solutions.drawing_utils cap = cv2.VideoCapture(0) with mp_face.FaceDetection(model_selection=0, min_detection_confidence=0.5) as face_det: while True: ok, frame = cap.read() if not ok: break rgb = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB) res = face_det.process(rgb) if res.detections: for d in res.detections: mp_draw.draw_detection(frame, d) cv2.imshow("MediaPipe Face Detection", frame) if cv2.waitKey(1) & 0xFF == ord("q"): break cap.release() cv2.destroyAllWindows()
