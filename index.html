Face Recognition Overview

Our system uses MediaPipe and Python to detect and track faces in real time and recognise people using stored face data.

Project Summary

MediaPipe is used for fast and efficient face detection, while OpenCV handles the webcam feed and drawing on frames. Recognition compares new faces against a dataset of known faces to identify people.

How It Works

1. Face Detection (MediaPipe)

  • Webcam frames are captured using OpenCV.
  • Each frame is processed with MediaPipe Face Detection.
  • MediaPipe locates the face area and key landmarks.
  • A box or overlay is drawn around the detected face.

2. Face Recognition

  • A dataset of known faces is prepared in advance.
  • Face encodings/features are stored for each person.
  • New detected faces are converted into encodings.
  • These are compared with stored encodings to find the best match.

Results

  • Real-time face detection works smoothly with normal lighting.
  • Tracking stays stable when the person moves in front of the camera.
  • Recognition accuracy is better with clear, front-facing training images.
  • Performance drops slightly in very low light or when the face is covered.
