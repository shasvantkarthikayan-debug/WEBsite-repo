Face Recognition Overview

Our system uses MediaPipe and Python to detect and track faces in real time and recognise people using stored face data.

See Working Code Overview View on GitHub

Project Summary

We use MediaPipe for fast and efficient face detection, OpenCV for webcam capture, and a dataset of known faces for recognition.

Python OpenCV MediaPipe AI

How It Works

1. Face Detection (MediaPipe)

  • Camera input captured with OpenCV.
  • Every frame processed with MediaPipe Face Detection.
  • MediaPipe finds face box + keypoints.
  • Box drawn around detected face.

2. Face Recognition

  • Dataset of known faces is created.
  • Encodings stored for each known person.
  • New face encoding compared with stored ones.
  • Best match = identity displayed.

import cv2 import mediapipe as mp mp_face = mp.solutions.face_detection mp_draw = mp.solutions.drawing_utils cap = cv2.VideoCapture(0) with mp_face.FaceDetection(model_selection=0, min_detection_confidence=0.5) as face_detection: while True: success, frame = cap.read() if not success: break rgb = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB) results = face_detection.process(rgb) if results.detections: for detection in results.detections: mp_draw.draw_detection(frame, detection) cv2.imshow("MediaPipe Face Detection", frame) if cv2.waitKey(1) & 0xFF == ord("q"): break cap.release() cv2.destroyAllWindows()

Results

  • MediaPipe detects faces instantly.
  • Tracking stays smooth even with movement.
  • Recognition improves with clear training images.
  • Low light reduces accuracy slightly.
